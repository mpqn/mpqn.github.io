手机阅读App竞品分析

发表于 2017-05-02 |

作业

一、 背景

据速途研究院2016年2月1日报告显示，2015年移动阅读市场整体规模达100.8亿元，同比增长近20个点。移动阅读市场稳步发展，增速放缓。移动互联网应用中阅读领域渗透率高达60%，可见阅读是目前最主流娱乐应用类型之一。根据易观智库发布的 《中国移动阅读市场趋势预测报告2016-2018》 数据预测，未来3年中国移动阅读市场规模增长幅度比较平稳，至2018年，这一数字将达到166.0亿元人民币，将会探索出更多元的商业模式。

二、 竞品对象

微信读书、网易蜗牛读书、掌阅iReader、多看

三、 竞品分析

3.1 定位和功能

3.1.1 产品定位

微信读书：出版图书/社区
蜗牛读书：出版图书/社区
掌阅iReader：出版图书、网络小说
多看阅读：出版图书、网络小说

3.1.2 产品功能

微信读书：微信读书是基于微信关系链的阅读应用。
在提供海量书籍资源和良好阅读体验的同时，融合了好友的阅读经历，通过微信好友的在读信息为你推荐优质书籍，支持与好友沟通互动，思想碰撞，交流想法。
为了增加阅读趣味性，微信读书还提供了与微信好友PK每周读书时长排行。

蜗牛读书：网易蜗牛读书，精选好书每天畅读一小时！
囊括优质出版书籍的阅读APP，每天享受畅读一小时特别福利，更有专业领读人入驻荐书，精品书评、优质书单伴你共读。

掌阅iReader：掌阅读书是一款专注于手机阅读领域的经典阅读软件。支持EBK3/TXT/UMD/EPUB/CHM/PDF全主流阅读格式。功能强大，个性时尚，界面简约，与各大出版社进行深度战略合作，拥有广阔图书资源。

多看阅读：多看阅读，国内卓越的移动阅读平台，以精美排版享誉全国，获2亿用户青睐！

3.2 视觉和风格

掌阅iReader：白色/轻量/扁平化/底部Tab栏

微信读书：白色/轻量/扁平化/底部Tab栏

蜗牛读书：白色/轻量/扁平化/底部Tab栏

多看阅读：白色/简洁/侧边栏

3.2.3 亮点功能和核心优势

掌阅iReader：

核心优势：内容全（目前拥有图书、杂志、漫画等类别的优质数字版权45万册，与500多家出版机构建立了合作，年发行图书15亿册，在国内移动阅读市场份额居于首位。）
丰富的图书格式支持

微信读书：

核心优势：微信一键登录；
基于熟人构建的阅读圈；
读书排行榜；
阅读时长可兑换书币购买书籍

蜗牛读书：

核心优势：名人领读；
丰富的书单；
付费灵活，全平台书籍任意读；
每天赠送一个小时阅读时长。

多看：

核心优势：专业排版；
云服务，本地书籍可上传至云端，并且支持同步阅读进度，书签和书摘，真正的多终端无缝体验，

3.3 运营模式

掌阅iReader：手机厂商合作、广告投放、签到送书币可购买书籍

微信读书：免注册（微信一键登录）、应用宝推荐、微信用户的传播（读书排行榜）、阅读时长兑换书币

蜗牛读书：免注册（网易帐号直接登录）、广告投放、每天免费内一小时阅读时长、每天低至一元钱就可以阅读全平台书籍、专业的书评和领读人吸引用户

多看阅读：小米手机预装、签到送书币

3.4 优缺点总结

产品 优点 缺点
掌阅iReader： 书全 、支持电子书格式最多，支持中文竖版阅读、翻页动画细腻逼真 不支持本地书籍备份云端
微信读书： 读书排行榜促进养成读书习惯、方便分享想法给朋友，阅读时长可以兑换书币买新书 不支持本地阅读、书少
蜗牛读书： 专业书评、书单让找书更方便，每天免费一小时阅读时长满足轻量阅读 不支持本地阅读、书少
多看阅读： 每日限免图书、排版精美、支持本地书籍备份云端，可同步书签、笔记 书少 、图书商城界面杂乱

四、总结

综合来看，如果想要看网络文学，并且阅读在线图书和本地图书兼而有之的话，多看和掌阅都是不错的选择；如果是轻量阅读用户，那么微信读书和蜗牛读书均是不错的选择。不得不说的是掌阅这些年在移动端的布局，奠定了今天的市场地位，对于微信读书和蜗牛读书来说，真正的差异化体验才是俘获用户的根本。
