Android Studio 快捷键

发表于 2017-02-18 | 分类于 技术 |

工欲善其事，必先利其器！
工作之余做了一些常用快捷键的总结，其中不免疏漏，不求全，但求好用！

Android快捷键

Ctrl + Shift

I 预览方法定义
T 环绕
M 抽取方法
C 抽取为常量
P 抽取为参数
A 快速进入指定设置
F8 断点设置
Enter 快速完成 （方法体大括号的添加、行尾的分号、自动格式化改行等操作）
Space 自动补全代码
/ 注释/解注释代码块

Ctrl + Alt

S 进入设置
F 抽取为全局变量
L 格式化代码
Space 类名或接口名提示
F12 文件路径（不常用）
V 生成返回值
H 找到方法的调用栈
B(=F4) 进入方法内部 等同于 ctrl+鼠标左键
W 代码，连续按会逐渐扩大范围
“[“/“]” 可以跳到大括号的开头/结尾

Ctrl

O 重载基类的方法
R 替换文本
F 查找文本
Q API文档参数注释
G 到某行某列（不常用）
F12 代码大纲 输入方法名可快速定位
F11 签添加标记
“+”/“-“ 展开或折叠 （针对当前行所在的代码块）
E 查看最近打开的文件
N 搜索类并编辑
D 当前行
Y 当前行
p 方法参数
J 代码模板
/ 注释/解注释代码行

Alt

shift + Up/Down 光标当前行
Insert 生成代码 (如get,set方法,构造函数等)
F1 代码所在位置
Up/Down 在方法间快速移动定位

Else

F2/Shift+F2 错误或警告快速定位
shift+F6 重命名
F11 快速添加或删除书签

以上所有快捷键均经过实际操作验证。 ——码字不易，转载请注明，谢谢！
